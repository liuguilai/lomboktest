@Property

Handy shortcut for generating @Getter and @Setter at once

Since

@Property was introduced as experimental feature in lombok v1.16.9

Experimental

Experimental because:

  • Still not sure that @Property will be approved.

Overview

There's pretty often a need in Java World to generate getters and setters for a class/property. For example when you're using some (ORM) frameworks. Where @Data is overhead (and actually might work in a wrong way when you need to use only some fields in equals/hashCode and toString methods) when you use it. And you might become pretty tired to write @Getter and @Setter every time.

For example when you have lot of data model classes you might become tired to write each time @Getter and @Setter.

Example:

  @Getter
  @Setter
  public class Person {
    private UUID id;
    private String firstName;
    private String lastName;
  }

Instead you might want to reduce you boilerplate code to something smaller like:

  @Property
  public class Person {
    private UUID id;
    private String firstName;
    private String lastName;
  }
